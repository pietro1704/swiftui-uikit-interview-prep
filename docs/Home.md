# 📚 Docs — SwiftUI + UIKit Interview Prep

A focused study companion for senior iOS engineers preparing for technical interviews.

## Pages

- ⚡️ **[4-Day Fast Track for Senior Engineers](Fast-Track-Senior.md)** — Dense schedule for engineers with 5+ years of experience
- ❓ **[50 Mock Interview Questions](Interview-Questions.md)** — Compact answers, ~3 min each
- 🧠 **[SwiftUI Cheat Sheet](Cheat-Sheet-SwiftUI.md)** — Property wrappers, modifiers, navigation
- 🛠 **[UIKit Cheat Sheet](Cheat-Sheet-UIKit.md)** — Lifecycle, Auto Layout, Compositional, Diffable
- ⚡️ **[Concurrency Cheat Sheet](Cheat-Sheet-Concurrency.md)** — async/await, actors, AsyncStream
- 🏛 **[Architecture Patterns](Architecture-Patterns.md)** — MVVM / VIPER / TCA / Coordinator — when to use each
- 🐛 **[Common Pitfalls](Common-Pitfalls.md)** — Bugs and gotchas that show up in interviews

## How to use

1. Clone and open the Xcode project (see [README](../README.md)).
2. Pick a study path (fast-track for seniors, or one-lesson-per-day).
3. For each lesson: read → interact in the simulator → solve the TODO.
4. Use the cheat sheets to refresh memory; rehearse with the mock questions.
